# CyberHub Website

A modern, high-performance educational website for Computer Science & Cyber Security students.

## Features

- **Study Materials**: Organized resources for C++, Data Structures, Networks, and more.
- **Online Code Editor**: Write and run code (simulated execution for non-JS languages) in the browser.
- **Cyber Tools**:
  - Base Converter (Binary/Decimal/Hex)
  - Bitwise Calculator
  - Logic Gate Helper
  - ASCII Lookup
- **Dark Mode UI**: tailored for late-night coding sessions.
- **Responsive Design**: Works on mobile and desktop.

## How to Run

1.  Simply open `index.html` in any modern web browser.
2.  No installation or server required.

## Technologies

- HTML5
- CSS3 (Custom Variables, Flexbox, Grid)
- JavaScript (Vanilla)
